​🎨 Trilha de CSS - Digital Innovation One

​Este projeto é uma Landing Page desenvolvida como parte dos desafios práticos da plataforma DIO (Digital Innovation One). O objetivo principal foi aplicar conceitos fundamentais e avançados de estilização com CSS puro, sem o uso de frameworks externos.
​

🚀 Tecnologias Utilizadas

​HTML5: Estruturação semântica do conteúdo.

​CSS3: Estilização avançada, incluindo:
​Flexbox & Grid: Para alinhamento e posicionamento dos elementos.
​Gradients: Uso de linear-gradient para fundos dinâmicos.
​Background Management: Manipulação de imagens de fundo com cover e center.
​Typography: Customização de fontes e sombras de texto (text-shadow).
- **Design Responsivo:** Implementação de Media Queries para adaptação em dispositivos móveis, tablets e desktops.
- **Flexbox & Grid:** Para layouts fluidos e alinhamento dinâmico.


​ ### 📋 Funcionalidades do Projeto
- **Responsividade Total:** Interface adaptável que garante uma boa experiência de usuário tanto em smartphones quanto em navegadores de desktop.
- **Banner Adaptável:** Ajuste automático de imagens e textos para diferentes proporções de visor.

💡 Aprendizados
​Durante o desenvolvimento deste projeto, foquei em:
​Centralização de elementos utilizando margin: auto e display: flex.
​Trabalho com camadas e posicionamento relativo/absoluto.
​Criação de botões e cards com border-radius e efeitos visuais.
- **Responsividade:** Entendimento profundo de como as `@media queries` e unidades flexíveis (como `flex-basis` e `clamp`) evitam quebras de layout em telas maiores.
- **Flexbox Avançado:** Uso de `flex-wrap` para permitir que elementos se reorganizem sozinhos conforme o espaço disponível.
